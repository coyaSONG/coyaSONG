## How I work

- Reproduce the problem before changing code, then add regression coverage.
- Follow each repository's own contribution, formatting, and validation workflow.
- Report environment limitations separately from failures caused by a change.
- Disclose substantive AI assistance and keep the verification evidence reviewable.
